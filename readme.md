Atlas is a web app that contains many games to test your geography knowledge. It is built with Blazor WebAssembly and .NET 8.0.

## Games

### [Random flag challenge](https://atlas.pulsewave.dev/flags/random)

Test your knowledge of flags by guessing the country of a random flag.
